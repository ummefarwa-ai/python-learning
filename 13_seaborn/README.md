# 📈 Seaborn

This folder contains my practice notebooks while learning **Seaborn**, a high-level Python library for statistical data visualization built on top of Matplotlib.

## 📚 Topics Covered

- Introduction to Seaborn
- Bar Plots
- Count Plots
- Histograms
- Box Plots
- Violin Plots
- Scatter Plots
- Heatmaps
- Pair Plots
- Real Dataset Visualization

## 📂 Files

| Notebook | Description |
|----------|-------------|
| 01_intro.ipynb | Introduction to Seaborn |
| 02_barplot.ipynb | Creating bar plots |
| 03_countplot.ipynb | Visualizing category frequencies |
| 04_histplot.ipynb | Distribution analysis with histograms |
| 05_boxplot.ipynb | Detecting outliers using box plots |
| 06_violinplot.ipynb | Distribution visualization with violin plots |
| 07_scatterplot.ipynb | Exploring relationships between variables |
| 08_heatmap.ipynb | Correlation matrices and heatmaps |
| 09_pairplot.ipynb | Visualizing relationships among multiple variables |
| 10_real_dataset.ipynb | Exploratory Data Analysis (EDA) using a real dataset |
